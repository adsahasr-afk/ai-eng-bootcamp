# AI Engineering Bootcamp

End-to-end AI engineering practice. Build real products, not prototypes.

## Structure
- `phase1-llm-foundations/` - LLM APIs + first end-to-end app (Weeks 1-2)
- `phase2-rag/`             - RAG architecture + evaluation (Weeks 3-4)
- `phase3-agents/`          - Agents + orchestration (Weeks 5-6)
- `phase4-llmops/`          - LLMOps, observability, deployment (Weeks 7-8)
- `capstone/`               - Capstone project
- `shared/`                 - Common utils
- `docs/`                   - Notes, architecture diagrams

## Setup
```bash
python3 -m venv .venv && source .venv/bin/activate
cp .env.example .env   # add keys locally, never commit
```
